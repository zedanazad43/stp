# Demo Playbook — Zero‑Cost Mode

## الهدف
تشغيل عرض حي يوصل القيمة بدون أي مزودات مدفوعة.

## الإعداد
1) انسخ `.env.deploy.example` إلى `.env.local`
2) اضبط:
   - DEMO_MODE=true
   - POLYGON_RPC_URL شبكة اختبار (Amoy/Mumbai)
   - STRIPE/PAYPAL وضع الاختبار فقط
3) تثبيت واعتماد:
```bash
pnpm install
pnpm dev
```

## سيناريو العرض (10 دقائق)
- الخطوة 1: تصفح الأرشيف وإظهار طابع تاريخي (بيانات نموذجية)
- الخطوة 2: سكّ NFT تجريبي (ميتاداتا على IPFS اختباري أو mock)
- الخطوة 3: عرض في السوق وإجراء شراء تجريبي (Mock Payment)
- الخطوة 4: إبراز سجل الملكية والتوثيق

## إيقاف كل ما قد يسبب كلفة
- شبكات testnet فقط
- تعطيل أي webhook حي
- عدم استخدام مفاتيح إنتاج لأي مزود

## نجاح العرض
- اكتمال رحلة المستخدم دون أخطاء
- إبراز الأمان والشفافية وسهولة الاستخدام
